# 2026-04-16 上下文记忆

## Vue 前端重构完成

使用 Vue 3 + TypeScript + Element Plus + Vite 重构了 WorldQuant Alpha Manager 的前端界面。

### 项目结构

```
frontend/
├── src/
│   ├── api/           # API 接口定义
│   │   ├── index.ts   # 基础 API 封装 (任务管理、回测、同步、提交)
│   │   └── database.ts # 数据库操作 API (Alpha、候选池、筛选)
│   ├── components/    # 公共组件
│   │   └── Layout.vue # 页面布局组件 (侧边栏+主内容)
│   ├── router/        # 路由配置 (7个页面)
│   ├── styles/        # 全局样式
│   │   └── index.css  # 自定义滚动条、统计卡片、表格样式等
│   ├── views/         # 页面视图
│   │   ├── AlphasView.vue     # Alpha 列表 (统计、搜索、筛选、详情)
│   │   ├── BacktestView.vue    # 批量回测 (进度、实时日志、配置)
│   │   ├── ResultsView.vue     # 回测结果 (统计、筛选、分页)
│   │   ├── TasksView.vue       # 任务列表 (状态监控、操作)
│   │   ├── FilterView.vue      # 筛选导出 (条件筛选、导出)
│   │   ├── SubmitView.vue      # Alpha 提交 (候选池、提交历史)
│   │   └── LogsView.vue        # 日志查看 (文件列表、内容)
│   ├── App.vue        # 根组件
│   └── main.ts       # 入口文件 (Pinia、Element Plus、Router)
├── index.html        # HTML 模板
├── package.json      # Vue 3.4、Element Plus 2.6、Vite 5.2
├── vite.config.ts    # Vite 配置 (代理 /api 到 localhost:5000)
└── tsconfig.json     # TypeScript 配置
```

### API Server 更新

在 `web/api_server.py` 中添加了：
1. **CORS 支持** - 允许跨域访问
2. **数据库 API**:
   - `GET /api/to_test_alphas` - 获取待回测列表
   - `GET /api/database/alphas` - 获取 Alpha 列表 (支持分页)
   - `GET /api/database/alpha/<id>` - 获取 Alpha 详情
   - `GET /api/database/tested_expressions` - 获取已回测表达式
   - `GET /api/database/candidates` - 获取候选池
   - `GET /api/database/stats` - 获取统计信息
   - `POST /api/database/alphas/search` - 搜索 Alpha

### 启动方式

1. **单独启动前端**: `start_frontend_vue.bat` (端口 3000)
2. **全栈启动**: `start_all_vue.bat` (API 5000 + Vue 3000)

### 技术选型理由

- **Vue 3 Composition API** - 更灵活的组件逻辑组织
- **Element Plus** - 成熟的 Vue 3 UI 组件库，与原 Streamlit 功能对应
- **Vite** - 快速的热更新和构建
- **TypeScript** - 类型安全，便于维护
- **Pinia** - 轻量级的状态管理 (预留)

### 注意事项

- API Server 需要先启动 (端口 5000)
- 前端默认代理 /api 到 localhost:5000
- 数据库 API 使用 sqlite3 直接查询
