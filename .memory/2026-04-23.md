### API 接口

- `GET /api/database/alphas/{alpha_id}` - 获取 Alpha 详情
- `POST /api/neutralization/test` - 单个 Alpha 中性化测试
- `POST /api/neutralization/batch-test` - 批量中性化测试

### 注意事项

- 批量测试耗时较长，建议选择较少的 Alpha 同时测试
- 可以设置并发数提高测试效率（最大 5）
- 中性化结果会自动标记优质 Alpha（★）
